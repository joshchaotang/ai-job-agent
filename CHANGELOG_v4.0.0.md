# AI Job Agent - 版本變更紀錄

## v4.0.0 Aurora Edition（2026-02-11）🎉 滿級分 UX 新手引導系統

### 重大更新

**全新新手引導體驗**
- ✨ 7 步驟互動式教學（歡迎畫面 → 完成）
- 💡 每一步都有清楚講解與視覺提示
- 🎯 智能 Spotlight 高亮目標元素
- 📊 進度條即時顯示完成度
- 🔄 可隨時跳過 / 重新開始
- 📱 完美支援手機與桌面

**引導流程**：
1. 📤 上傳履歷（支援 PDF、Word、TXT）
2. 🔑 設定 DeepSeek API Key（免費申請指引）
3. 🔍 搜尋適合的職缺（關鍵字匹配）
4. 🤖 AI 智能分析（4 維度評分）
5. 📊 查看分析結果（雷達圖 + 建議）
6. 🔔 設定推送通知（Email/Telegram/Slack，選填）
7. 🎉 完成設定，開始使用！

**UX 優化**：
- 🎨 Glassmorphism 毛玻璃引導框
- ⚡ 流暢動畫與過渡效果
- 💬 簡潔易懂的說明文字（不超過 2 句）
- 🎮 遊戲化體驗（進度點、徽章、動畫）
- 🆘 右下角永久幫助按鈕（💡）
